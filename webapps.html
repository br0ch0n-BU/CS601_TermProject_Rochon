Downstairs bath remodel

This one was lots of work! I budgeted a few weeks but it ended up taking me six months. The downstairs bath is the one that guests see, so I had a grand vision of what I wanted it to look like. The main goal was for it to match the faux-craftsman motif of the rest of the house.

This took so long because there were many hidden tedious steps like rerouting plumbing and electrical on top of the normal remodel work. I ended up buying lots of new tools.

There were many steps. I'm probably forgetting some:

  1. Scrape up old tile floor from concrete.
  2. Tear out old fiberglass shower enclosure.
  3. Remove old giant mirror glass.
  4. Remove old vanity.
  5. Remove old toilet.
  6. Grind off old thinset from floor.
  7. Cut and re-do shower drain to align with new shower floor dimensions.
  8. Cut out moldy old drywall from many places.
  9. Remove old lighting.
  10. Remove old medicine cabinet.
  11. Re-route poorly routed electrical in wall framing.
  12. Move switches and electrical higher to account for taller new vanity.
  13. Prime and pour self-leveling compound to smooth out concrete floor.
  14. Reroute sink water supply to fit new vanity.
  15. Install new Wedi shower pan and wallboard.
  16. Install new drywall where needed.
  17. Tile and grout room and shower floors.
  18. Tile and grout walls.
  19. Remove and replace exhaust fan.
  20. Install new medicine cabinet.
  21. Install crown moulding.
  22. Reroute vanity lighting electrical.
  23. Caulk and paint everything.
  24. Install new toilet flange.
  25. Install new toilet.
  26. Make new P-trap for taller vanity.
  27. Install new vanity.
  28. Install new vanity mirror.
  29. Install new vanity lighting.
  30. Install new shower hardware.
